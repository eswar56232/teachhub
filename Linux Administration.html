Linux Administration

In this course, you will learn the following topics:

Introduction to Linux and Open Source
Understand the basics of Linux, its history, and the principles of open source software.
Installing Linux (Distributions, Dual Boot, Virtualization)
Learn about different Linux distros, installation methods including dual boot and virtualization.
Linux Filesystem and Directory Structure
Explore the Linux filesystem hierarchy and the purpose of key directories.
User and Group Management
Manage users, groups, and permissions to control access on a Linux system.
File Permissions and Security
Understand Linux file permissions, ownership, and basic security practices.
Process Management and Job Scheduling
Monitor and control running processes and schedule jobs using cron and at.
Package Management and Software Installation
Install and manage software packages using tools like apt, yum, or dnf.
Networking in Linux (Configuring Network Interfaces, DNS, etc.)
Configure network settings, interfaces, and DNS on Linux systems.
Disk Management and File Systems
Manage disks, partitions, and filesystems including mounting and formatting.
Backup and Recovery in Linux
Learn backup strategies and recovery tools to protect data.
System Monitoring and Performance Tuning
Use tools to monitor system resources and optimize performance.
Shell Scripting and Automation
Write shell scripts to automate repetitive tasks and improve efficiency.
System Logging and Log Management
Configure and analyze system logs for troubleshooting and auditing.
Security and Firewall Configuration
Set up firewalls and implement security best practices on Linux.
Remote Administration (SSH, SFTP)
Manage Linux systems remotely using SSH and secure file transfers with SFTP.
Linux System Services and Daemons
Understand and manage background services and daemons on Linux.
Troubleshooting and Debugging
Learn methods and tools for diagnosing and fixing Linux system issues.
Linux System Updates and Patch Management
Keep Linux systems updated and secure with patch management techniques.
Final Project
Apply all learned concepts to set up and manage a fully functional Linux system.
