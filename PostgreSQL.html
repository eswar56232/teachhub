PostgreSQL

In this course, you will learn the following topics:

  • Introduction to PostgreSQL
  • Installing and Setting Up PostgreSQL
  • PostgreSQL Architecture and Tools (psql, pgAdmin)
  • Data Types and Constraints in PostgreSQL
  • Creating and Managing Databases and Tables
  • Basic SQL Queries in PostgreSQL
  • Functions and Operators (Text, Math, Date, etc.)
  • Joins, Subqueries, and Set Operations
  • Data Manipulation (INSERT, UPDATE, DELETE)
  • Indexes and Performance Optimization
  • Views and Materialized Views
  • Transactions and Concurrency (ACID, MVCC)
  • Stored Procedures and Functions (PL/pgSQL)
  • Triggers and Event Handling
  • User Roles and Permissions (Security)
  • Backup, Restore, and Maintenance
  • Advanced Data Types (JSON, Arrays, HSTORE)
  • Partitioning and Table Inheritance
  • Extensions and Full-Text Search
  • PostgreSQL in Real-World Applications (Web, APIs, Analytics)

Estimated Time to Complete: 8–10 weeks

Prerequisites: Basic understanding of programming (helpful but not required)
